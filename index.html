The Application

This repository builds an application which is a derivation of the simon. The application is written in c++ using qt 5.5 libraries to ensure cross-platform compatibility

Compiling

The easiest way to compile the application is to use the QtCreator ide and open the Qt project file. You should then be able to compile and run the code inside Qt Creator using a valid c++ compiler
